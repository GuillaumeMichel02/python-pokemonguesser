Petite application pour s'entrainer à connaître les noms des pokémons. Fait pour Windows avec Python 3.10

Entrez le nom du pokémon affiché puis tapez Entrée pour valider. Si c'est correct, l'application passe à l'image suivante. Sinon, le vrai nom est affiché et il faut réappuyer sur Entrée pour continuer.

Si un pokémon est correctement deviné 3 fois (ou 4 je sais plus), il n'apparaîtra plus tant que l'application reste ouverte. Ecrire s puis Entrée dans la zone de texte enlève automatiquement le pokémon actuel de la liste.
